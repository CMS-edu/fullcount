권장:

- PNG 또는 WebP를 써도 되지만, 현재 코드는 위 이름의 PNG를 먼저 찾습니다.
- 투명 배경 정사각형 이미지가 제일 깔끔합니다.
- 대략 512x512 이상이면 충분합니다.
- 파일이 없으면 팀 색과 약칭 배지가 대신 나옵니다.
- 현재 `assets/HT.png`, `assets/SS.png` 같은 기존 로고 파일도 예비 경로로 연결되어 있습니다. 위 새 파일명으로 넣으면 새 파일이 우선 표시됩니다.
